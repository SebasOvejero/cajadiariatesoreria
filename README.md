# 🏥 Clinical SaaS - Sistema de Gestión de Órdenes Médicas

Un dashboard clínico ultra-moderno y optimizado para la gestión de transacciones y órdenes médicas. Este sistema elimina la complejidad de los esquemas de facturación tradicionales por sesiones, implementando un modelo de **precio único y cerrado por orden médica**, ideal para centros de salud y consultorios independientes que buscan agilidad y claridad financiera.

Diseñado bajo estándares de diseño premium (inspirado en la estética de Stripe y Linear) con un enfoque prioritario en la experiencia de usuario (UX/UI).

---

## ✨ Características Principales

* **Modelo de Facturación Simplificado:** Eliminación total del concepto de "sesiones/días". El precio se establece una sola vez por orden médica, reduciendo errores humanos en la carga de datos.
* **Diseño UX/UI Premium (SaaS Clinical):** Interfaz limpia con una paleta de colores basada en verdes esmeralda clínicos (`emerald/teal`), fondos pizarra suaves (`slate-50`) y tipografía de alta legibilidad.
* **Header Glassmórfico:** Barra de navegación superior flotante con efectos de desenfoque de fondo (`backdrop-blur-md`) que optimiza el espacio de trabajo.
* **Administrador de Obras Sociales Integrado:** Configuración ágil de aranceles fijos y coberturas por entidad.
* **Control de Caja Diario:** Registro fluido de movimientos, estados de pago y desglose rápido de aranceles.
* **Métricas Operacionales de un Vistazo:** Reportes automatizados que calculan la cantidad de órdenes totales y el **Valor Promedio por Orden** en lugar de promedios por sesión.
* **Formatos Listos para Auditoría:** Exportación y visualización de reportes optimizados para el control administrativo.

---

## 🚀 Tecnologías Utilizadas

El proyecto está construido utilizando tecnologías modernas que permiten un renderizado rápido, portabilidad y nula necesidad de configuración pesada de servidores:

* **HTML5 & JavaScript (ES6+):** Lógica del lado del cliente fluida, manejo de estados reactivos para las pestañas y el filtrado de transacciones.
* **Tailwind CSS (Vía CDN):** Framework de diseño utilitario para lograr componentes responsivos, transiciones suaves y la estética SaaS premium.
* **Lucide Icons / FontAwesome:** Iconografía vectorial minimalista y adaptativa.

---

## 🛠️ Estructura del Proyecto

El sistema está diseñado como una **Single Page Application (SPA)** contenida en un archivo optimizado, lo que facilita su despliegue inmediato.
